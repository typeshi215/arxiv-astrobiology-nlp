## 🌟 What is arxiv-astrobiology-nlp?

arxiv-astrobiology-nlp is a software tool designed to analyze and explore topics from a large collection of astrobiology research papers. It works by modeling themes from over 52,000 abstracts on ArXiv, spanning research published between 1996 and 2025. This helps users understand emerging trends and key areas in astrobiology using two natural language processing (NLP) methods: BERTopic and Top2Vec.

You don’t need any programming skills to use this application. It provides an easy way for researchers, students, and enthusiasts to discover scientific topics quickly without reading all papers.

## 📊 How This Works (Background for Interested Users)

This tool applies natural language processing algorithms to cluster research papers into topics based on their abstracts. BERTopic and Top2Vec are two well-known methods in the NLP field for creating meaningful groups of documents. By comparing both, users can see differences in topic detection. The processed data comes from a scientific archive called arXiv, which publishes preprints in many research fields, here filtered for astrobiology.

This pipeline and dataset are designed to reproduce scientific findings with transparency, supporting research communities interested in topic modeling and scientometrics.
